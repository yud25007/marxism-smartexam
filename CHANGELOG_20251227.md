# CHANGELOG - 2025-12-27

## 安全与权限加固
- **游客身份隔离机制 (Guest Sandbox)**：
    - 修复了“极速试用”结束后游客临时 VIP 身份残留的漏洞。
    - **自动销毁**：点击“首页”或返回主界面时，系统将自动检测并清除游客 Session，确保临时权限不被带入正式练习区域。
    - **存储拦截**：游客的练习数据不再同步至云端数据库，实现纯净的内存级体验。
- **强类型校验补全**：
    - 为所有临时用户对象注入了强制性的 `status: 'ACTIVE'` 字段，解决了组件渲染时的潜在类型未定义风险。

## 新功能与体验优化
- **AI 权限与模型管理增强**：
    - 管理员控制台新增 **Gemini 3 Flash** 模型支持，满足更快速、低成本的解析需求。
    - 补全了 `authService` 层的模型切换链路，实现从数据库到 UI 的完整配置闭环。

- **全卷笔记系统加固**：
    - **点击即同步**：新增“整理并存入笔记”时的实时数据库同步逻辑，确保学习成果零丢失。
    - **布局紧凑化**：优化了笔记卡片的 HTML 结构，消除了折叠列表间的冗余空行。
- **后台管理效能提升**：
    - 在管理员控制台新增“直达云端数据库”快捷入口，支持一键跳转至 Supabase 数据编辑器。
- **双向多活跳转 (Bi-directional Multi-link)**：
    - **主/镜像对称**：不仅支持主站跳转镜像，镜像站现在也能智能识别自身状态并引导用户跳回主站（Zeabur）。
    - **视觉感知增强**：根据当前所在域名动态切换提示语及配色（主站显示红色警示加速，镜像站显示蓝色稳健提示），大幅降低用户在多活架构下的迷失感。
- **UI 智能响应优化**：
    - 进一步细化了加速球的显示逻辑，确保其在“维护模式”下依然作为唯一的逃生出口保持高亮可见。

## 故障修复
- **权限逃逸修复**：关闭了游客通过“首页”按钮间接获取所有锁定题库访问权限的逻辑路径。
- **重复数据过滤**：防止了 `guest-session` 产生的冗余记录污染 `exam_results` 统计报表。
