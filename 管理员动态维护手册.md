# 马原在线答题系统：管理员动态维护手册

本手册指导管理员如何使用新增的云端管理工具，进行题库维护、系统配置及公告发布。

---

## 🛠 一、 核心控制中心 (System Control)
在管理员后台顶部，您可以实时切换系统的全局状态：
*   **允许云端同步**：开启后，方可点击下方的“同步题库”按钮。
*   **系统维护模式**：用于临时关闭答题功能。
*   **允许自主注册**：控制登录页面的注册入口开关。

## 📚 二、 题库云端化维护 (Cloud Questions)
系统已实现从“静态代码”向“动态数据库”的迁移。

### 1. 首次数据同步
如果您更新了代码（如 Git Pull 了最新的 `extra_data.ts`），请点击：
**[同步本地题库到云端]**
> 注意：此操作会根据 ID 覆盖云端数据，请谨慎操作。

### 2. 在线修改标准答案
*   **后台修改**：在“题库管理”模块选择章节，直接在列表中点击“铅笔”图标修正答案。
*   **即时修正**：在“详细解析”页面查看错题时，管理员可直接使用题目下方的快捷工具修正答案，无需切换页面。

## 📝 三、 富文本笔记与 LaTeX (Rich Text)
*   **所见即所得**：全卷笔记现已支持通过工具栏直接改变文字颜色、高亮及划线。
*   **公式支持**：AI 解析会自动渲染 LaTeX 公式，如 $\frac{c}{v}$ 形式的资本有机构成公式将以专业排版显示。
*   **快速删除**：在解析界面点击红色垃圾桶图标，可精准移除对应题目的笔记。

## 📢 四、 精准身份公告 (Announcement)
发布新公告时，您可以选择：
*   **目标身份**：设置为 `VIP` 或 `MEMBER`，实现定向通知。
*   **历史回顾**：用户点击图标可查看历史公告，弹窗仅弹出最新的未读内容。

---

## 🎨 五、 界面文案个性化修改 (Text Customization)
如果您需要修改网页上显示的文字（如“注册已禁用”的提示），请参考以下文件路径：

### 1. 注册与登录相关 (`components/AuthViews.tsx`)
*   **注册禁用提示**：搜索 `REGISTRATION_DISABLED` 附近的文字。
*   **登录框标题**：搜索 `登录账号`。
*   **注册申请提交成功**：搜索 `注册申请已提交`。

### 2. 维护界面相关 (`App.tsx`)
*   **Windows 更新文案**：在文件末尾的 `Global Maintenance Interceptor` 区域，可以修改 `正在准备理论同步`、`天涯若比邻` 等文字。

### 3. 公告弹窗相关 (`components/Announcement.tsx`)
*   **今日不再提示**：搜索该关键词即可修改。

> **修改技巧**：建议使用 VS Code 的“全局搜索”功能 (Ctrl + Shift + F)，直接搜索您想改的那段文字，即可精准定位代码行。

---

## 🚀 六、 云端数据库直达链接 (Database Direct Access)
为了方便高级运维人员直接操作底层数据，系统在后台新增了直达入口：
*   **位置**：管理员控制台顶部的“云端模式已启用”横幅内。
*   **功能**：一键跳转至 Supabase 数据编辑器，支持直接执行 SQL 指令或批量导出数据。
*   **注意**：在数据库控制台操作时请务必确认操作范围，建议操作前先通过 Supabase 面板进行数据备份。

---

建议每周检查一次“待审核申请”，并在题库同步完成后关闭“允许同步”开关以增强安全性。
