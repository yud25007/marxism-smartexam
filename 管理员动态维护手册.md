# 马原在线答题系统：管理员动态维护手册

本手册指导管理员如何使用新增的云端管理工具，进行题库维护、系统配置及公告发布。

---

## 🛠 一、 核心控制中心 (System Control)
在管理员后台顶部，您可以实时切换系统的全局状态：
*   **允许云端同步**：开启后，方可点击下方的“同步题库”按钮。
*   **系统维护模式**：用于临时关闭答题功能。
*   **允许自主注册**：控制登录页面的注册入口开关。

## 📚 二、 题库云端化维护 (Cloud Questions)
系统已实现从“静态代码”向“动态数据库”的迁移。

### 1. 首次数据同步
如果您更新了代码（如 Git Pull 了最新的 `extra_data.ts`），请点击：
**[同步本地题库到云端]**
> 注意：此操作会根据 ID 覆盖云端数据，请谨慎操作。

### 2. 在线修改标准答案
*   **后台修改**：在“题库管理”模块选择章节，直接在列表中点击“铅笔”图标修正答案。
*   **即时修正**：在“详细解析”页面查看错题时，管理员可直接使用题目下方的快捷工具修正答案，无需切换页面。

## 📝 三、 富文本笔记与 LaTeX (Rich Text)
*   **所见即所得**：全卷笔记现已支持通过工具栏直接改变文字颜色、高亮及划线。
*   **公式支持**：AI 解析会自动渲染 LaTeX 公式，如 $\frac{c}{v}$ 形式的资本有机构成公式将以专业排版显示。
*   **快速删除**：在解析界面点击红色垃圾桶图标，可精准移除对应题目的笔记。

## 📢 四、 精准身份公告 (Announcement)
发布新公告时，您可以选择：
*   **目标身份**：设置为 `VIP` 或 `MEMBER`，实现定向通知。
*   **历史回顾**：用户点击图标可查看历史公告，弹窗仅弹出最新的未读内容。

---
**维护建议**：
建议每周检查一次“待审核申请”，并在题库同步完成后关闭“允许同步”开关以增强安全性。
